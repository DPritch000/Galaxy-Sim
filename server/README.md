# Local Server

This folder contains a minimal Node.js static server for the Galaxy Sim project.

## Run

1. Open a terminal in this `server` folder.
2. Start the server:
   ```bash
   npm start
   ```
3. Open:
   - http://localhost:3000

## Optional port

You can choose a custom port.

### PowerShell

```powershell
$env:PORT=8080
npm start
```

Then open http://localhost:8080
